Pagina Web

En este laboratorio desarrollamos un sitio web informativo que nos permite organizar y presentar todas las actividades realizadas hasta el momento. Para lograrlo, utilizamos la inteligencia artificial Genspark, capaz de generar páginas completas a partir de prompts y del material que le proporcionamos previamente.

Con esta herramienta planificamos el diseño y la estructura que deseábamos para el sitio, integrando explicaciones, recursos visuales y secciones pensadas para distintos tipos de usuarios. El objetivo fue obtener una página clara, accesible y visualmente coherente con el resto del proyecto.

Además, parte del material audiovisual generado se incorporó directamente a esta página. A continuación se incluye un espacio preparado para cargar un video desde un archivo físico, sin depender de plataformas externas.
